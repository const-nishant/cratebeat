# Changelog

All notable changes to **CrateBeat** will be documented in this file.

This project follows [Conventional Commits](https://www.conventionalcommits.org/) and semantic versioning.

---

## [0.1.0] - 2025-11-18
### Added
- Initial project structure and module layout
- Terminal UI using ratatui + crossterm
- ASCII banner and grid layout rendering
- Real-time sequencer engine with BPM control
- Manual drum pad triggering (a/s/d/f)
- Audio loader and playback via rodio + cpal
- Basic pattern editor (toggle steps using space/arrow keys)
- MIT License, README, gitignore
